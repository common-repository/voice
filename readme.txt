== Installation ==

This section describes how to install the plugin and get it working.

1. Search for `Voice` in the WP-Admin plugin install screen.
1. Click `Install` and then click the `Activate` button.
1. That’s it! Voice comes with a default configuration and you’re now ready to start creating new Voice Blocks or add voice capabilities to your existing WordPress blocks!
1. Create a new Voice Block from the Add New Block menu in the WordPress editor.
1. In the editor, click on ANY block, and then click the `transform icon` to create a new Voice Block. Voice blocks support multiple blocks so you can create Voice Groups!
